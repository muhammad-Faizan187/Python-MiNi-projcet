🎓 Python Mini Database System Guide
This is a simple and beautiful Python based Teacher Record Management System.
It helps teachers manage student records easily using Python.
━━━━━━━━━━━━━━━━━━ ✨ HOW TO USE ━━━━━━━━━━━━━━━━━━
🔹 Run the Python program.
After running, you will see these options:
1️⃣ Add New Student
2️⃣ View All Students
3️⃣ Search Student
4️⃣ Delete Student
5️⃣ Total Students
6️⃣ Exit Program
━━━━━━━━━━━━━━━━━━ 📌 FEATURES EXPLAINED ━━━━━━━━━━━━━━━━━━
✅ 1. Add New Student
Enter student name and student ID.
The program saves the data in the database.
Example:
👤 Name → Ali
🆔 ID → 1023
━━━━━━━━━━━━━━━━━━
✅ 2. View All Students
Shows all saved student records with names and IDs.
━━━━━━━━━━━━━━━━━━
✅ 3. Search Student
Find any student record quickly by entering student name.
━━━━━━━━━━━━━━━━━━
✅ 4. Delete Student
Remove any student record from the database.
━━━━━━━━━━━━━━━━━━
✅ 5. Total Students
Displays the total number of saved students.
━━━━━━━━━━━━━━━━━━
✅ 6. Exit Program
Closes the program safely.
